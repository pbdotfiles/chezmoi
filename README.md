# `chezmoi` dotfiles

System setup and dotfiles using [`chezmoi`](https://www.chezmoi.io/)

## Quick start

1. Install chezmoi: `sh -c "$(curl -fsLS get.chezmoi.io)"` in `~`
2. Run `~/bin/chezmoi init https://github.com/pb4git/chezmoi.git`
3. Run `~/bin/chezmoi apply`
