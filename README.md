# `chezmoi` dotfiles

System setup and dotfiles using [`chezmoi`](https://www.chezmoi.io/)

## Quick start

1. Install git.
2. Install `chezmoi`.
3. Run `chezmoi init https://github.com/pb4git/chezmoi.git`
4. Run `chezmoi apply`
